# {% include icon.html icon="fa-solid fa-users" %}研究团队

同济大学“人工智能+大气海洋”实验室的研究团队主要由穆斌教授、袁时金教授及他们指导下的多名博士生和硕士生组成。当前主持国家自然科学基金重点联合基金项目一项、国家自然科学基金面上项目一项、国家重点研发计划课题两项、以及上海市科委重点课题一项。团队在该研究方向已发表学术论文120余篇，获授国家发明专利11项。
